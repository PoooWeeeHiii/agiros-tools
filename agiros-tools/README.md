# AGIROS Tools

Command line utilities for AGIROS packaging and build workflows.

## Install (wheel)

```bash
python -m build /Users/wangruiqi/Desktop/AGIROS_TOOLS/packages/agiros-tools
pip install /Users/wangruiqi/Desktop/AGIROS_TOOLS/packages/agiros-tools/dist/agiros_tools-0.1.0-py3-none-any.whl
```

## Commands

- `agiros-tools-menu`
- `agiros-oob-build`
- `agiros-tracks-download`
- `agiros-release-download`
- `agiros-git-build`
- `agiros-rpm-build`
- `agiros-clean-generated`
